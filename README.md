# W-festiva
Interface completa do W Festival: streaming ao vivo, line-up de artistas, conteúdo exclusivo e Watch Again. Desenvolvida em React + Vite + Tailwind com foco em UX: carrosséis arrastáveis, animações suaves, menu lateral por gesto no mobile e layout responsivo. Projeto de portfólio demonstrando boas práticas de front-end e design de produto.
